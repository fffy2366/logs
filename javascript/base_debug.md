# JavaScript基础之调试方法总结

## 1. 最简单的方法alert()

alert() 是最简单的调试方法，缺点是会中断程序执行，需要手动关闭弹出窗口。

## 2. 控制台打印console.log()

console.log() 这种方法可以在浏览器debug的控制台输出。

## 3. 浏览器调试设置断点

以chrome为例，菜单->更多工具->开发者工具下的sources下，找到需要调试的源码，在行号处单击即可设置断点，然后刷新页面即可进入
